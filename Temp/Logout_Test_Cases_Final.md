## Notes:
- All test cases use tester name: **TanVD**
- Logout functionality:
  - Customer logout: Clears CustomerID, CustomerName, Role from session
  - Admin/Employee logout: Clears EmployeeID, EmployeeName, Role, RoleID, BranchId from session
  - All logout actions redirect to Home/Index
  - Authentication cookies are removed based on scheme (CustomerScheme, AdminScheme, EmployeeScheme)
- Logout links:
  - Customer: "Đăng Xuất" in avatar dropdown menu (top right)
  - Admin: "Đăng xuất" in sidebar menu (bottom)
  - Employee: "Đăng xuất" in sidebar menu (bottom)
- Multiple authentication schemes can coexist (CustomerScheme, AdminScheme, EmployeeScheme)
- Logout only affects the current user's session, not other users
- After logout, users cannot access protected pages and must log in again
